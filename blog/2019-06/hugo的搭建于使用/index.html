hugo的搭建于使用

hugo的本地搭建

  1. 安装hugo：使用sudo apt-get install hugo安装hugo，之后使用hugo version查看hugo版本确认安装是否成功。
  2. 新建文件夹：使用hugo new site mysite生成一个新的博客文件夹。
  3. 安装主题：前往hugo主题网站选择主题。通过git下载到mysite/themes/。可通过mysite路经下的config.toml文件修改主题配置。
  4. 启动博客：使用hugo server -t hugo-theme-learn --buildDrafts可以在本地启动博客。通过访问localhost:1313。
  5. 新建文章：使用hugo new location/name.md新建文章。删除文章可直接删除文件。

hugo的远端部署

  1. 新建仓库：在github新建名为name.github.io的仓库。
  2. 在博客文件夹下生成public文件夹：使用hugo --theme=hugo-theme-learn --baseUrl="https://name.github.io/" --buildDrafts生成public文件夹。
  3. 关联远端仓库：转到刚生成的public文件夹里，使用git init，git add .，git commit -m "备注",git remote add origin https://github.com/name/name.github.io.git关联远端仓库。
  4. 推送至远端：使用git push -u origin master命令开始推送，输入帐号和密码即可。
